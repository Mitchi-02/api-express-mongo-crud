# Crud API with Express and MongoDB
A crud api built with Express and MongoDB, for a school assignement.
## Getting Started
### Clone the Repository
  ```bash
  git clone https://github.com/Mitchi-02/api-express-mongo-crud.git
  ```
### Run Seeders
```bash
  npm run seed
  ```
### Start the server
```bash
  npm install
  npm run dev
  ```
